# Aeko Program

Use the Aeko Program Crate to write on-chain programs in Rust.  If writing client-side applications, use the [Aeko SDK Crate](https://crates.io/crates/aeko-sdk) instead.

More information about Aeko is available in the [Aeko documentation](https://aeko.com/docs).

[Aeko Program Library](https://github.com/aeko-labs/aeko-program-library) provides examples of how to use this crate.

Still have questions?  Ask us on [Stack Exchange](https://sola.na/sse)
