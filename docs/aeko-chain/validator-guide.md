# Validator Guide

Validators secure AEKO Chain and earn rewards.

---

## Requirements

- High-performance CPU
- SSD storage
- Stable internet
- Minimum AEKO stake

---

## Responsibilities

- Produce blocks
- Validate transactions
- Maintain uptime
- Participate in governance

---

## Slashing Conditions

- Double signing
- Extended downtime
- Invalid state transitions

---

## Rewards

- Block rewards
- Transaction fees
- Governance influence
