# Gas and Fee Model

AEKO Chain uses a **compute-unit–based fee system**.

---

## Fee Formula

Total Fee = Base Fee + (Compute Units × CU Price)


---

## Compute Units

Each instruction consumes compute units based on:
- CPU usage
- Memory access
- Cryptographic operations

---

## Fee Markets

- Dynamic pricing during congestion
- Priority fees supported
- Predictable cost model for developers

---

## Fee Distribution

- Validator rewards
- Network treasury
- Security incentives
