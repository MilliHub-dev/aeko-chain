# Incident Response

In the event of a security breach or network halt.

## Emergency Powers
The **Security Council** (9 elected experts) has limited emergency powers:
1.  **Zone Freeze**: Pause a specific smart contract or bridge.
2.  **Validator Kick**: Eject a malicious validator from the active set.
*   *Constraint*: Powers expire after 72 hours unless ratified by the Token House.

## Response Plan
1.  **Detection**: Monitoring systems alert the Council.
2.  **Triage**: Council confirms the severity.
3.  **Containment**: Execute Zone Freeze if necessary.
4.  **Remediation**: Develop and deploy a patch.
5.  **Post-Mortem**: Publish a full report to the community.
