# Reputation System

The **AEKO Reputation Score** is a global, on-chain metric of trustworthiness.

## Calculation

Your score (0-1000) is calculated based on:

1.  **Identity Verification**: +Points for L1/L2/L3 clearance.
2.  **Account Age**: +Points for longevity.
3.  **Staking**: +Points for staking AEKO (Skin in the game).
4.  **Behavior**:
    *   +Points for consistent posting and genuine interaction.
    *   -Points for being reported/blocked by high-reputation users.
    *   -Severe Penalty for spamming or scamming (Slashing).

## Usage

*   **Feed Ranking**: High-reputation users appear higher in comments and feeds.
*   **Governance**: Only high-reputation users can vote on critical protocol upgrades.
*   **Airdrops**: Projects can target "Real Humans" by filtering for Score > 500.
