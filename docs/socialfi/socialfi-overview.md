# SocialFi Overview

**SocialFi** (Social Finance) is the fusion of social media and decentralized finance. AEKO Chain is built from the ground up to support this ecosystem, making social interactions (likes, shares, comments) transactable events.

## The Shift

| Web2 Social | AEKO SocialFi |
| :--- | :--- |
| **Data** | Owned by Platform | Owned by User (On-Chain) |
| **Monetization** | Ads (Platform keeps 100%) | Tips/Subs (Creator keeps 95%+) |
| **Influence** | Opaque Algorithms | Transparent Reputation Score |
| **Identity** | Siloed (Twitter login) | Portable (Wallet login) |

## Core Primitives

1.  **Tokenized Content**: Posts as NFTs.
2.  **Tokenized Influence**: Creator Coins.
3.  **Programmable Feeds**: Algorithms you can audit and choose.
