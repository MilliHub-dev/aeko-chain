# Contributing to AEKO Chain

We welcome contributions from the community! AEKO Chain is open-source and community-driven.

## Getting Started

1.  **Fork the Repo**: Click "Fork" on GitHub.
2.  **Clone**: `git clone https://github.com/YOUR_USERNAME/aeko-chain.git`
3.  **Branch**: Create a feature branch (`git checkout -b feature/amazing-feature`).

## Development Environment

You will need:
*   Rust (1.75+)
*   Solana CLI Tools
*   Node.js (18+)

## Submitting a PR

1.  Ensure all tests pass (`cargo test`).
2.  Run the linter (`cargo fmt` and `cargo clippy`).
3.  Open a Pull Request describing your changes in detail.
4.  Link any relevant Issues (e.g., "Fixes #123").
