Aeko Explorer API
Purpose

The Explorer API powers:

AekoScan

Dashboards

Analytics

Compliance tools

Endpoints
| Endpoint              | Description    |
| --------------------- | -------------- |
| `/blocks`             | Latest blocks  |
| `/txs`                | Transactions   |
| `/accounts/{address}` | Account info   |
| `/tokens`             | Token registry |
| `/nfts`               | NFT registry   |
| `/posts`              | Social posts   |

Example
GET /accounts/0x123
