# Staking

Staking secures the AEKO Chain and earns rewards.

## Validator Staking
*   **Minimum Stake**: 100,000 AEKO (Self-bond).
*   **Hardware**: 12-core CPU, 128GB RAM, 1TB NVMe (High performance required).
*   **Rewards**: Block rewards + Transaction fees.

## Delegated Staking
Users can delegate their AEKO to a validator.
*   **Minimum**: 1 AEKO.
*   **Lock-up**: 3 days (Warm-up) / 3 days (Cool-down).
*   **APY**: ~5-8% (Dynamic based on total staked).

## Liquid Staking
AEKO supports native liquid staking via the `StakePool` program.
*   Deposit `AEKO` -> Receive `stAEKO`.
*   Use `stAEKO` in DeFi while earning staking rewards.
